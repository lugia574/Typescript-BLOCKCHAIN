# Typescript

## 개요

이것은 Typescript를 배우는것을 기록하기 위한 깃

타입스크립트를 쓰는 이유는

⭐ Javascript

└ 매우 유연해서 에러를 잘 보여주지 않음

🚫 숫자 배열 + false

→ 배열이 사라지고 string으로 바뀜

🚫 함수의 인자가 잘못 들어가도 실행됨

→ return값이 NaN일 뿐, 에러가 나지 않음

🚫 const a = { a: "A" };

a.hello();

실행 시 에러 발생

→ 실행 전에 에러 감지 불가

⭐ Typescript

└ 타입 안정성 → 버그 감소
